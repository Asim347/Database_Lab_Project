                                                     🚗 Smart Vehicle Management System
📌 Project Description

The Smart Vehicle Management System is a database-based project designed to manage vehicle, owner, and registration data in an organized way. It ensures efficient storage, retrieval, and management of records using structured database concepts.

🎯 Objective

The main objective of this project is to reduce manual record handling and provide a simple system for managing vehicle-related information. It improves accuracy, organization, and data accessibility.

🛠️ Technologies Used

This project is developed using database design concepts and SQL for data management. Documentation is prepared using Microsoft Word and exported in PDF format for submission.

📂 Project Contents

The project includes proposal documentation, milestone reports, and database design work. These files represent the planning and development stages of the system.
